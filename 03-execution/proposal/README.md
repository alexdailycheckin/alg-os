# Proposal Skill

Post-discovery client document. Generated from a discovery transcript or notes. Produces the lean client-facing proposal.

Customer-as-hero throughout. Voice rules applied. Pricing anchored against the operator's stated comparison (often hiring cost, not software). Length target 800 to 1,500 words. The next step is one specific low-commitment action, not "schedule a 60-minute follow-up."

See `foundation.md` section 5.3.
