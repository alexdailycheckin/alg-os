# Teardown Skill

Competitor and positioning analysis. Produces specific, actionable positioning gaps rather than summary descriptions of competitors.

Reads the operator's positioning, ICP, and brand files. Scrapes the named competitor's public surface (website, LinkedIn, public reviews). Categorises every claim into measurable / defensible / marketing buckets. Produces a head-to-head against the operator's win axis. Surfaces 3 to 5 exploitable weak spots and a single-sentence wedge.

See `foundation.md` section 5.3.
