# Artefact Skill

The signature artefact generator. Takes an operator's business and helps them design THEIR signature artefact. Not a template. A diagnostic.

Looks for the high-ticket, service-heavy, customer-facing pattern by default. Where the business does not fit, proposes alternative artefact types (state-of-category reports, anonymised benchmark studies, industry diagnostic tools).
